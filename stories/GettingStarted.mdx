# LogoMakerJS: Highly customizable sequence logos in JS and React

LogoMakerJS is a React component that renders sequence logos. It is highly customizable, allowing you to change colors, axes, labels to render logos for DNA, RNA or proteins. Logos are rendered as Scalable Vector Graphics (SVG) for high quality display on any device.

## Features

- **Many Supported Formats**: LogoMakerJS supports Position Probability Matrices (PPMs), Position Weight Matrices (PWMs), FASTAs and BigWig-like data.
- **Display Options**: Display logos as frequency, information content, or raw values.
- **Highly Customizable**: Customize colors, axes, labels, and more.
- **Add Annotations**: Add annotations to your logos to highlight important features.

See the [gallery](?path=/docs/gallery-dna-logos--docs) for examples of logos you can create with LogoMakerJS, read the `Logo` component [API documentation](?path=/docs/logo--docs) for more information on how to use the library, or read the tutorial to learn more.

## Quick Start

### Using React

LogoMakerJS is available as an npm package. It has minimal dependencies, only requiring React.

#### Installation

```
npm install logomakerjs
```

### Using plain JavaScript

#### Installation

The LogoMakerJS library is also available as a standalone JavaScript file. You can include it in your HTML file using a script tag. You will need to include React and ReactDOM scripts earlier in the file.

```html
<script
  crossorigin
  src="https://unpkg.com/react@18/umd/react.production.min.js"
></script>
<script
  crossorigin
  src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"
></script>
<script
  src="https://cdn.jsdelivr.net/npm/logomakerjs@0.1.2/dist/index.js"
  type="text/javascript"
></script>
```

#### Usage

The library exposes a global object `logomakerjs` that you can use to render logos. The following code embeds the DNA logo in a `div` element:

````html
<!doctype html>
<html>

    <script
      crossorigin

    ></script>

      crossorigin
      src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"
    ></script>
    <script
      src="https://cdn.jsdelivr.net/npm/logomakerjs@latest/dist/index.umd.js"
      type="text/javascript"
    ></script>
    <div id="logo" style="width: 500px"></div>
    <script type="text/javascript">
      const CTCF_PPM = [
        [0.09, 0.31, 0.08, 0.5],
        [0.18, 0.15, 0.45, 0.2],

        [0.04, 0.57, 0.36, 0.01],

        [0.93, 0.01, 0.03, 0.01],
        [0.0, 0.0, 0.99, 0.01],
        [0.36, 0.0, 0.64, 0.0],
        [0.05, 0.01, 0.55, 0.37],

    </script>
  </body>
````
